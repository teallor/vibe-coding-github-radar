# Code of Conduct

## Our commitment

We are committed to a respectful, constructive, and harassment-free project for everyone. Be considerate of different experience levels, languages, and viewpoints.

## Expected behavior

- Discuss ideas and evidence, not personal characteristics.
- Give actionable feedback and accept correction gracefully.
- Respect privacy and never publish another person's credentials or personal information.
- Avoid harassment, threats, discriminatory language, and disruptive behavior.
- Represent project status and adoption honestly.

## Enforcement

Report conduct concerns privately to the maintainer through an appropriate private GitHub channel. Public issues should not expose personal or sensitive details. The maintainer may edit, remove, or reject contributions and restrict participation when necessary to protect the community.

This code applies in repository spaces and when someone officially represents the project.
